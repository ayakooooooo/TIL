```
// 変数lengthを定義してください  
let length = 5;  

// 変数lengthの値を出力してください  
console.log(length);  

// 変数lengthを用いて、円の面積を出力してください  
console.log (length * length * 3 );  
```
***
変数は、一度代入した値を変更することもできる
一度値を代入した変数に、その後再び値を代入すると、上書きされる
「let」は必要なく 「変数名 = 新しい値」
***
```
let number = 7;
console.log(number);

// 変数numberの値に3を加えてください
number=number+3;

console.log(number);

// 変数numberの値を2で割ってください
number=number/2;

console.log(number);
```
