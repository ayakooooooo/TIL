# 関数  
いくつかの処理をまとめたもの
「あいさつをする」「名前を言う」という2つの処理を「introduce」という名前でまとめる、など  
```
function()  {
まとめたい処理
}
``` 
「関数を定義する」という 

関数を定義した後、  
```定数名();  ```
と書くことで関数の中の処理を実行できる  
「関数を呼び出す」という  
```
const greet = function() {
  console.log("こんにちは！");
  console.log("関数を学習していきましょう！");
};

// 関数を呼び出してください
greet();
```
# 関数の代入  
```
// 定数helloに関数を代入してください
const hello = function(){
console.log("こんにちは！");
console.log("私の名前は○○です");
};
// 定数helloに代入された関数を呼び出してください
hello();
```
